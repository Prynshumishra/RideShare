🚗 RideShare – Intelligent Carpooling Platform

Smart. Sustainable. Social.
RideShare is a full-stack ride-sharing platform that intelligently connects people traveling in the same direction — making commuting affordable, efficient, and eco-friendly 🌱

🌐 Live Application: https://ride-share-pm.vercel.app/


✨ Core Highlights
🚘 Dual Role System – Be a Driver, Passenger, or both

📍 Smart Matching Engine – Connects users based on routes

⚡ Instant + Scheduled Booking – Travel your way

💬 Real-Time Communication – Chat before confirming rides

⭐ Trust System – Ratings & reviews for transparency

👤 User Profiles – Personalized ride experience

🧠 How It Works

Sign Up / Login

Create or Search Ride

Match with Nearby Routes

Connect via Chat

Book & Travel Together 🚗

🛠️ Tech Architecture
🎨 Frontend

| Technology   | Purpose                 |
| ------------ | ----------------------- |
| React.js     | Build dynamic UI        |
| Context API  | Global state management |
| React Hooks  | State & lifecycle       |
| React Router | Navigation              |
| Axios        | API communication       |
| Tailwind CSS | Utility-first styling   |
| ShadCN UI    | Modern UI components    |
| Lucide Icons | Clean icon system       |

⚙️ Backend
| Technology | Purpose             |
| ---------- | ------------------- |
| Node.js    | Runtime environment |
| Express.js | API server          |
| MongoDB    | Database            |
| Mongoose   | ODM for MongoDB     |
| JWT        | Authentication      |
| bcryptjs   | Password security   |
| Multer     | File uploads        |
| Cloudinary | Image hosting       |
| dotenv     | Environment config  |
| cors       | Secure API access   |
